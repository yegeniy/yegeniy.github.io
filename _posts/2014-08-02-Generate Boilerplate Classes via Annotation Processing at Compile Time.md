Use an annotation and an accompanying Annotation Processor, to generate Java
classes from a Mustache template at compile time. Useful for generating
boilerplate classes.

This topic is a little larger than a snippet allows, due to the amount of wiring
required. Please refer to the [`java_snippet_2014-08-02` repository](https://github.com/yegeniy/java_snippet_2014-08-02/)
for extended details and example code.
